# PrimaryKeyType {#reference164 .reference}

PrimaryKeyType specifies the type of the primary key.

## Enumeration data type {#section_jyr_mtq_dfb .section}

-   INTEGER: Integer

-   STRING: String

-   BINARY: Binary


```language-protobuf
enum PrimaryKeyType {
    INTEGER = 1;
    STRING = 2;
    BINARY = 3;
}

```
